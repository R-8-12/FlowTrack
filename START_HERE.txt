  QUICK START GUIDE
================================================================================

STEP 1: Install MySQL (if not already installed)
-------------------------------------------------
Download MySQL 8.0 from:
https://dev.mysql.com/downloads/installer/

During installation:
- Choose "Developer Default"
- Set root password: root (or remember your password)
- Keep default port: 3306

STEP 2: Create Database
------------------------
Double-click: setup-mysql-database.bat

This will create the 'ims_db' database automatically.

STEP 3: Run Application
------------------------
Double-click: test-and-run.bat

The application will:
1. Connect to MySQL
2. Create all tables automatically
3. Insert sample data (first run only)
4. Start on http://localhost:8087

STEP 4: Login
-------------
Open browser: http://localhost:8087

Default credentials:
Username: admin
Password: admin123

================================================================================
  CONFIGURATION
================================================================================

Database Settings (application.properties):
- Database: ims_db
- Host: localhost
- Port: 3306
- Username: root
- Password: root

To change password:
Edit: src/main/resources/application.properties
Change: spring.datasource.password=YOUR_PASSWORD

================================================================================
  BACKUP YOUR DATA
================================================================================

Create Backup:
mysqldump -u root -p ims_db > backup.sql

Restore Backup:
mysql -u root -p ims_db < backup.sql

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "Access denied"
Solution: Check password in application.properties

Problem: "Unknown database"
Solution: Run setup-mysql-database.bat

Problem: "Connection failed"
Solution: Start MySQL service
  - Open Services (services.msc)
  - Find MySQL80 or MySQL
  - Click Start
